## 業務描述

DraftKings 是美國最大的純線上博彩平台，專注於線上運動博彩 (Online Sports Betting, OSB) 和在線賭場遊戲 (iGaming)。公司採用 B2C 商業模式，直接面向消費者提供數字博彩服務，無實體賭場或零售業務。

**核心業務線**:
1. **Sportsbook (運動博彩)**: 提供多種體育賽事投注，包括 Same Game Parlays (SGP)、Live Betting 等創新產品
2. **iGaming (在線賭場)**: DK Casino 品牌，提供老虎機、桌遊等線上賭場遊戲
3. **Daily Fantasy Sports (DFS)**: 日常夢幻體育遊戲，公司起家業務
4. **新產品**: DraftKings Predictions (即將推出的預測遊戲產品)

## 競爭地位

### 主要競爭對手

1. **BetMGM (MGM Resorts + Entain 50/50 合資)**
   - 比較: 盈利能力領先 (首家分紅 $100M+), 但規模小於 DKNG
   - DKNG 優勢: 規模更大 (Q3 $1,047M vs BetMGM 未完整披露)
   - BetMGM 優勢: 已實現盈利和分紅，MGM 零售資產協同

2. **Caesars Digital (Caesars Entertainment)**
   - 比較: 第二大線上博彩營收 ($311M Q3), 但僅為 DKNG 的 30%
   - DKNG 優勢: 規模優勢 (3.4x Caesars)
   - Caesars 優勢: 6000萬+ Caesars Rewards 會員基礎，全渠道整合

3. **Penn Entertainment (ESPN BET, theScore Bet)**
   - 比較: 全渠道模式，42 個零售物業，iGaming 高增長 (+41%)
   - DKNG 優勢: 純線上規模優勢
   - Penn 優勢: 全渠道客戶價值 6x，theScore 媒體引流

4. **FanDuel (Flutter Entertainment)**
   - 比較: 市占率與 DKNG 不相上下的主要競爭對手
   - 市場地位: 雙寡頭格局

### 競爭優勢

1. **最大規模優勢**: Q3 線上博彩營收 $1,047M，是 Caesars 的 3.4 倍
2. **全國品牌認知度**: 純線上模式中的領導品牌
3. **產品創新**: Same Game Parlays, Live Betting, 即將推出 Predictions
4. **現金流改善**: 9M FCF $237M (+$223M YoY)，證明商業模式可行性
5. **運營靈活性**: 無零售業務包袱，可快速擴張到新州

### 競爭劣勢

1. **缺乏零售協同**: Penn 全渠道客戶價值 6x，DKNG 純線上獲客成本較高
2. **盈利能力落後**: BetMGM 已分紅，DKNG 仍虧損
3. **缺乏媒體資產**: Penn 有 theScore 媒體 (400萬 MAU)，DKNG 需依賴付費獲客
