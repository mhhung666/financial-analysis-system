### 暴跌原因拆解

#### 🔴 **主因1：Q4營收指引低於市場預期**
```
收盤前市場預期：Q4營收 $1,380M (YoY +18%)
盤後公司指引：Q4營收 $1,313-1,338M (YoY +15%)

市場反應：單日暴跌-20%
隱含市值蒸發：約$45億
```

**市場為何如此恐慌？**
- Q4是假日季，通常是全年最強季度（貢獻30-35%營收）
- 指引保守暗示：廣告主預算縮減或競爭加劇
- 增長放緩趨勢明確：18% → 17% → 15% 📉

#### 🔴 **主因2：美加核心市場增長停滯**
```
美加MAU增長：+4% (接近天花板)
美加ARPU增長：+5% (變現遇瓶頸)
美加營收增長：+9% (遠低於國際市場+41%)

美加佔總營收：75%
意涵：75%的營收增速僅9%，拖累整體
```

#### 🔴 **主因3：增長故事的可持續性存疑**
```
Q3實際：營收+17% YoY ← 表現尚可
Q4指引：營收+15% YoY ← 趨勢放緩
市場擔憂：2026年增長可能降至12-14%

估值調整：
- 前：P/S 5.2x (基於18%增長)
- 後：P/S 4.2x (基於15%增長)
跌幅-20%合理反映預期下修
```

#### ⚠️ **次要因素**
- **競爭加劇**：Google Shopping AI化，Instagram加強視覺搜索
- **宏觀不確定性**：廣告預算削減，消費者支出放緩
