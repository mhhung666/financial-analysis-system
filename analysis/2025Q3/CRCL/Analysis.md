## 公司概況

### 業務模式

Circle Internet Group 是全球領先的穩定幣和區塊鏈金融服務提供商，主要業務包括:

1. **USDC (USD Coin)**: 與美元 1:1 掛鈎的穩定幣，100% 儲備金支持
2. **Reserve Income**: 透過投資 USDC 儲備金獲取利息收入
3. **Circle Payments Network (CPN)**: 基於 USDC 的企業支付基礎設施
4. **USYC**: 代幣化貨幣市場基金
5. **Arc Network**: 正在開發的企業級 Layer-1 區塊鏈

### 營收模式

Circle 的營收主要來源於:
- **Reserve Income** (~90%): USDC 儲備金投資於短期美國國債和現金產品的利息收入
- **Other Revenue** (~10%): 包括交易費、CPN 費用、API 使用費、USYC 管理費等

### 價值主張

**對於用戶**:
- 穩定的價值儲存 (1 USDC = 1 USD)
- 快速、低成本的跨境支付
- 24/7 即時結算
- 程式化的支付和資金管理

**對於企業**:
- 降低跨境支付成本 (相比傳統銀行)
- 改善資金流動性
- 接入全球加密資產市場
- 透過 CPN 簡化集成

### RLDC Margin 深入分析

**RLDC (Revenue Less Distribution Costs)** 是 Circle 的核心利潤率指標:

```
RLDC = 總營收 - Distribution Costs
RLDC Margin = RLDC / 總營收
```

| 項目 | Q3 2025 | Q3 2024 | 變化 |
|------|---------|---------|------|
| 總營收 | $740M | $447M | +$293M |
| Distribution Costs | $454M | $278M | +$176M |
| RLDC | $286M | $169M | +$117M |
| RLDC Margin | 39% | 41% | -270bps |

**Margin 壓縮分析**:

1. **分銷成本增長較快**: +63% vs 營收增長 +66%
   - 顯示合作夥伴分潤增加
   - 可能反映競爭加劇，需要更高的激勵來獲取流通量

2. **QoQ 改善**: Margin 從 37% (Q2) 提升至 39% (Q3)
   - 顯示短期趨勢改善
   - 可能受益於合作夥伴組合優化或議價能力提升

3. **管理層評論**:
   - 長期目標維持 RLDC margin 在 40% 以上
   - Q3 的 39% 接近目標範圍

| **Circle** | USDC | ~29% | 透明度、合規性、美國監管 | 市占率較小、分銷成本高 |
| **PayPal** | PYUSD | <1% | 品牌、用戶基礎 | 新進者、生態系統小 |
| **Ripple** | RLUSD | 未推出 | 跨境支付網路 | 尚未推出 |

#### 競爭優勢分析

**Circle (USDC) 的優勢**:

1. **透明度與信任**:
   - 每月儲備證明報告
   - 獨立審計
   - 100% 儲備承諾
   - 優於 Tether 的透明度

2. **監管合規**:
   - NY DFS 監管
   - 符合 Circle Reserve Fund 框架
   - 為 GENIUS Act 等立法做好準備

3. **網路擴張**:
   - 支援 28 個區塊鏈網路
   - CCTP 跨鏈協議
   - 與主要 DeFi 協議集成

4. **企業級產品**:
   - Circle Payments Network
   - API 與 SDK
   - 白標解決方案

5. **Arc Network**:
   - 潛在的差異化優勢
   - 企業級區塊鏈基礎設施
   - 與傳統金融機構合作

**Tether (USDT) 的優勢**:

1. **網路效應**: 70% 市占率創造強大的流動性優勢
2. **深度集成**: 與所有主要交易所和 DeFi 協議深度集成
3. **新興市場**: 在拉美、非洲、亞洲新興市場佔主導地位
4. **交易量**: 日均交易量遠超 USDC

#### 競爭態勢評估

**市場動態**:

1. **雙寡頭格局**: USDT + USDC 佔據 99% 市場份額
2. **USDC 追趕**: 市占率從 23% 提升至 29%，顯示追趕勢頭
3. **監管催化劑**: 未來監管可能有利於更合規的 USDC
4. **新進者威脅有限**: PayPal, Ripple 等新進者短期內難以撼動雙寡頭格局

**競爭策略**:

Circle 的競爭策略聚焦於:
1. 透明度和合規性差異化
2. 企業級產品和服務
3. 創新 (Arc Network, CPN, USYC)
4. 與傳統金融機構合作

## 管理層討論

### CEO Jeremy Allaire 的戰略願景

在 Q3 財報電話會議上，CEO Jeremy Allaire 闡述了 Circle 的長期願景:

**三層架構**:

1. **經濟作業系統 (Economic OS)**:
   - Arc Network - 企業級 Layer-1 區塊鏈
   - 可程式化、安全、高性能
   - 為金融應用設計

2. **數位資產層**:
   - USDC - 數位美元
   - USYC - 代幣化貨幣市場基金
   - 未來可能擴展至其他資產

3. **應用實用工具**:
   - Circle Payments Network - 支付基礎設施
   - CCTP - 跨鏈轉帳協議
   - API & SDK - 開發者工具

### Arc Network 戰略

**核心目標**: 成為「Internet Financial System」的基礎設施

**差異化特點**:

1. **企業級設計**:
   - 高吞吐量、低延遲
   - 確定性交易確認
   - 企業級安全和合規功能

2. **Permissionless but Regulated**:
   - 無需許可的參與 (任何人都可以運行節點)
   - 但符合監管要求 (KYC/AML 在應用層)

3. **Native USDC Integration**:
   - USDC 作為原生 gas token 或主要資產
   - 降低使用門檻

4. **Arc Token**:
   - 正在探索原生代幣推出
   - 用途: 治理、質押、激勵
   - 經濟模型尚未確定

**進展與時程**:

- ✅ 2025年10月28日: 公開測試網啟動
- ✅ 100+ 企業參與測試網
- 🔄 2026年上半年: 主網候選版本 (預計)
- 🔄 2026年下半年: 主網正式啟動 (預計)
- 🔄 2026年或之後: Arc Token 推出 (可能)

### Circle Payments Network (CPN) 戰略

**目標**: 成為全球企業支付的首選基礎設施

**進展指標**:

| 指標 | 當前狀態 | 增長 |
|------|----------|------|
| 已註冊 FI | 29 家 | - |
| 審核中 FI | 55 家 | - |
| 管道中 FI | 500+ 家 | - |
| 年化 TPV | $3.4B | 101x 月度增長 |

**成功案例**:

1. **Brex**: 企業支出管理平台，使用 CPN 進行跨境支付
2. **Deutsche Börse**: 歐洲最大證券交易所，探索代幣化證券結算
3. **Itaú**: 拉美最大銀行之一，為客戶提供 USDC 服務
4. **Fireblocks**: 數位資產託管平台，集成 CPN

**擴張策略**:

1. **地理擴張**:
   - 北美、歐洲、拉美、亞洲
   - 重點: 美國、歐盟、巴西、新加坡、香港

2. **垂直行業**:
   - 跨境支付
   - 企業資金管理
   - 供應鏈金融
   - 薪資支付

3. **合作夥伴策略**:
   - 與銀行合作 (白標解決方案)
   - 與金融科技公司合作 (API 集成)
   - 與區塊鏈平台合作 (互操作性)
