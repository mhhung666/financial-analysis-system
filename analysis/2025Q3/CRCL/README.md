## 投資亮點

### ✅ 優勢

1. **市場領導地位穩固**: USDC 市占率從 23% 提升至 29%，在穩定幣市場持續擴大份額
2. **強勁盈利能力**: 調整後 EBITDA 利潤率達 57%，淨利潤年增 202%
3. **Arc Network 生態系統**: 獲得頂級金融機構支持，有潛力成為企業級區塊鏈基礎設施
4. **CPN 快速成長**: 101倍的 TPV 月度增長展現強大的網路效應
5. **法規環境改善**: GENIUS Act 等立法進展有利於穩定幣業務發展

### ⚠️ 挑戰

1. **股價大幅回調**: 從 2025年6月高點 $263.45 跌至 11月的 $76.94 (-71%)
2. **RLDC 利潤率壓縮**: 年減 270bps，反映競爭加劇和分銷成本上升
3. **Arc Token 不確定性**: 代幣經濟模型尚未明確，執行風險存在
4. **監管風險**: 雖然環境改善，但穩定幣監管仍在演進中
5. **市場競爭**: Tether (USDT) 仍佔據 70% 市場份額
