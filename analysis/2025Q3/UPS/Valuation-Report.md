## 估值方法三：DCF 現金流折現法

### WACC 計算

```
【WACC 參數】
無風險利率 (Rf): 4.35% (US 10Y Treasury, 2025-11-14)
Beta: 0.85 (Yahoo Finance / Bloomberg 平均)
市場風險溢酬 (MRP): 6.5%
權益成本 (Re) = 4.35% + 0.85 × 6.5% = 9.88%

債務成本 (Rd): 4.5% (加權平均債券收益率)
稅率: 24% (有效稅率)
Rd (稅後) = 4.5% × (1-24%) = 3.42%

資本結構:
權益 (E): $107,724M (85.6%)
淨債務 (D): $18,018M (14.4%)
總資本: $125,742M

WACC = 85.6% × 9.88% + 14.4% × 3.42%
     = 8.46% + 0.49%
     = 8.95% ≈ 9.0% (基準假設)
```

### 自由現金流預測 (5年模型)

**計算公式**:
- NOPAT = (營業利潤 - D&A) × (1 - 稅率) + D&A = EBIT × (1 - 稅率) + D&A × 稅率
- FCF = NOPAT - CapEx - NWC 變化

**明確預測期 FCF 現值總計**: $19,009M

### 終值計算

```
2030 FCF: $5,600M
永續增長率 (g): 2.5% (假設與美國長期 GDP 增長一致)

終值 = FCF₂₀₃₀ × (1+g) / (WACC - g)
     = $5,600M × 1.025 / (9.0% - 2.5%)
     = $5,740M / 6.5%
     = $88,308M

終值現值 = $88,308M × 0.596
         = $52,632M
```
