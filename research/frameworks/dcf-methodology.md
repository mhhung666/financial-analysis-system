## 🎯 DCF 估值概述

### 什麼是 DCF？
**折現現金流（Discounted Cash Flow, DCF）** 是一種內在價值評估方法，透過：
1. 預測公司未來自由現金流
2. 用適當折現率折現至現值
3. 計算企業內在價值

### 為什麼使用 DCF？
- ✅ 基於基本面，不受市場情緒影響
- ✅ 靈活性高，可調整各種假設
- ✅ 強迫深入思考業務模式
- ⚠️ 對假設非常敏感（GIGO原則）
- ⚠️ 需要大量財務知識

## 📋 DCF 估值步驟

### Step 1: 預測自由現金流（FCF）

#### 1.1 選擇 FCF 類型

**FCFF (Free Cash Flow to Firm)**：
```
FCFF = EBIT × (1 - 稅率) + 折舊攤銷 - 資本支出 - 營運資金變動
```
適用：所有公司

**FCFE (Free Cash Flow to Equity)**：
```
FCFE = 淨利 + 折舊攤銷 - 資本支出 - 營運資金變動 ± 淨借款變動
```
適用：無槓桿或低槓桿公司

**簡化版（常用）**：
```
FCF = 經營現金流 - 資本支出
```

#### 1.2 歷史 FCF 分析

| 年份 | 營收 | EBITDA | FCF | FCF Margin |
|------|------|--------|-----|------------|
| 2023 | $XXX | $XXX | $XXX | XX% |
| 2024 | $XXX | $XXX | $XXX | XX% |
| 2025E | $XXX | $XXX | $XXX | XX% |

**分析重點**：
- FCF 是否穩定增長？
- FCF Margin 趨勢如何？
- 資本支出佔營收比例？

#### 1.3 預測未來 FCF（5-10年）

**營收預測**：
- Year 1-3：明確階段（根據指引、產業趨勢）
- Year 4-5：漸進階段（增長率遞減）
- Year 6+：穩態增長（GDP增長率）

**範例預測**：
```
2026E: 營收 $XX億 (+15%) → FCF $XX億 (margin 30%)
2027E: 營收 $XX億 (+13%) → FCF $XX億 (margin 31%)
2028E: 營收 $XX億 (+11%) → FCF $XX億 (margin 32%)
2029E: 營收 $XX億 (+9%)  → FCF $XX億 (margin 33%)
2030E: 營收 $XX億 (+7%)  → FCF $XX億 (margin 33%)
```

---

### Step 2: 計算折現率（WACC）

#### 2.1 WACC 公式

```
WACC = (E/V) × Re + (D/V) × Rd × (1 - 稅率)

其中：
E = 股權市值
D = 負債市值
V = E + D (企業總價值)
Re = 股權成本
Rd = 負債成本
```

#### 2.2 計算股權成本（Re）- CAPM模型

```
Re = Rf + β × (Rm - Rf)

其中：
Rf = 無風險利率（10年期美債殖利率，約4-5%）
β = Beta值（股票相對市場波動度）
Rm - Rf = 股權風險溢酬（約7-9%）
```

**範例計算**：
```
假設：
- Rf = 4.5%（10年期美債）
- β = 1.2（科技股）
- 風險溢酬 = 8%

Re = 4.5% + 1.2 × 8% = 14.1%
```

#### 2.3 計算負債成本（Rd）

```
Rd = 利息支出 / 平均負債

或使用：
Rd = Rf + 信用利差
```

#### 2.4 WACC 範例計算

```
假設公司：
- 市值：$100億（股權E）
- 負債：$20億（D）
- 總值：$120億（V）
- Re = 14.1%
- Rd = 5%
- 稅率 = 21%

WACC = (100/120) × 14.1% + (20/120) × 5% × (1-0.21)
     = 11.75% + 0.66%
     = 12.4%
```

#### 2.5 常用 WACC 範圍

| 公司類型 | 典型WACC |
|----------|----------|
| 成熟大型股 | 8-10% |
| 成長型科技股 | 10-12% |
| 高成長小型股 | 12-15% |
| 高風險公司 | 15%+ |

---

### Step 3: 折現現金流至現值

#### 3.1 折現公式

```
PV = FCF / (1 + WACC)^n

n = 年數
```

#### 3.2 範例計算表

| 年份 | FCF預測 | 折現係數 | 現值PV |
|------|---------|----------|--------|
| 2026 | $100M | 0.890 | $89M |
| 2027 | $113M | 0.792 | $90M |
| 2028 | $125M | 0.705 | $88M |
| 2029 | $136M | 0.627 | $85M |
| 2030 | $145M | 0.558 | $81M |
| **明確期合計** | - | - | **$433M** |

---

### Step 4: 計算終值（Terminal Value）

#### 4.1 永續增長法（Gordon Growth Model）

```
Terminal Value = FCF(最後一年) × (1 + g) / (WACC - g)

g = 永續增長率（通常 2-3%，不超過GDP增長）
```

**範例**：
```
假設：
- 2030年FCF = $145M
- WACC = 12.4%
- 永續增長率 g = 3%

TV = $145M × (1 + 3%) / (12.4% - 3%)
   = $149.35M / 9.4%
   = $1,589M

折現至現值：
PV(TV) = $1,589M / (1.124)^5
       = $1,589M × 0.558
       = $887M
```

#### 4.2 出場倍數法（Exit Multiple）

```
Terminal Value = EBITDA(最後一年) × 出場倍數

出場倍數參考同業平均 EV/EBITDA
```

---

### Step 5: 計算企業與股權價值

#### 5.1 企業價值（Enterprise Value）

```
EV = 明確期現金流現值 + 終值現值

範例：
EV = $433M + $887M = $1,320M
```

#### 5.2 股權價值（Equity Value）

```
Equity Value = EV + 現金 - 負債

範例：
股權價值 = $1,320M + $250M(現金) - $0(負債)
         = $1,570M
```

#### 5.3 每股內在價值

```
內在價值/股 = 股權價值 / 流通股數

範例：
內在價值 = $1,570M / 100M股
         = $15.70/股
```

## ⚠️ DCF 常見陷阱

### 1. 過度樂觀的增長假設
❌ **錯誤**：假設高增長永續（20%+ for 10年）
✅ **正確**：增長率隨時間遞減至永續水平

### 2. 忽略週期性
❌ **錯誤**：用高峰期 FCF 作基準
✅ **正確**：使用正常化/平均 FCF

### 3. 永續增長率過高
❌ **錯誤**：g = 5-6%（超過名目GDP）
✅ **正確**：g = 2-3%（保守估計）

### 4. WACC 計算錯誤
❌ **錯誤**：使用過低折現率
✅ **正確**：根據公司風險合理設定

### 5. 忽略資本需求
❌ **錯誤**：只看EBITDA，忽略Capex
✅ **正確**：使用真實 FCF（扣除Capex）

## 💡 實務建議

### 何時使用 DCF？
✅ **適合**：
- 穩定現金流業務（SaaS, 消費品）
- 成熟期公司
- 需要長期持有決策

❌ **不適合**：
- 初創公司（無穩定現金流）
- 週期性極強產業（原物料）
- 快速變化產業（估計困難）

### 如何提高準確度？
1. **保守假設**：寧可低估不高估
2. **情境分析**：熊市/基準/牛市三種情境
3. **敏感度測試**：測試關鍵變數影響
4. **歷史驗證**：回測歷史DCF準確度
5. **結合其他方法**：與P/E、P/S交叉驗證

## 📝 DCF 快速檢查表

建立 DCF 模型時檢查：

### 輸入假設
- [ ] 歷史財務數據準確（3-5年）
- [ ] 營收增長率合理且遞減
- [ ] FCF margin 基於歷史趨勢
- [ ] Capex 假設符合業務需求
- [ ] 營運資金變動合理

### 折現率
- [ ] 無風險利率使用當前10年美債
- [ ] Beta 值來源可靠
- [ ] WACC 計算正確
- [ ] 折現率符合公司風險

### 終值
- [ ] 永續增長率 ≤ 3%
- [ ] 或出場倍數符合產業平均
- [ ] 終值不超過總值 70-80%

### 結果檢驗
- [ ] 隱含P/E、P/S 是否合理
- [ ] 與同業估值比較
- [ ] 敏感度分析涵蓋合理範圍
- [ ] 三種情境都有計算
