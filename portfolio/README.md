# Portfolio Directory

This directory organizes portfolio tracking by year.

## Structure

```
portfolio/
├── 2025/              # Current year portfolio data
│   ├── holdings.md    # Current positions
│   ├── performance.md # Performance tracking
│   ├── risk-matrix.md # Risk assessment
│   └── watchlist.md   # Stocks to watch
├── 2024/              # Previous year archives
└── README.md          # This file
```

## Usage

- Each year has its own directory for historical tracking
- The current year contains active portfolio management files
- Archive previous years at year-end for historical reference
